# Fixer

Fix file and folder permission error.

修复文件和文件夹的权限错误。

## Install

`go install github.com/svtter/fixer`

## HOWTO

Just run `fixer` under the project path.

#### example 1

```bash
cd <the directory want to be fixed.>
fixer
```

#### example 2

```bash
fixer the directory want to be fixed.>
```
